# Arduino



## Getting started

First of all please download & install Arduino IDE from official page.
Then, you can edit .ino files.

## Run the code as ros node

```
cd <your workspace>
(source devel/setup.bash)
rosrun rosserial_python serial_node.py _port:=/dev/ttyACM0
```
